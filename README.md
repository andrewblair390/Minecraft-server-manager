
# What is this?
This is a lightweight server manager for my own personal use. It will allow a minecraft server to automatically restart and not much else right now.

# How do I use this?
Place the script directly into the folder with the server jar and run it. From there you can set the memory to launch with and view the console/send commands. The main server Jar must be named "server.jar". All other serverside configuration must be done by editing server-properties.
